# Painel auxiliar de extração de mods em .scs & .zip
**Extractor base:** https://github.com/sk-zk/Extractor


## Como usar
1) Arraste seu arquivo ou mod em .scs ou .zip para cima do SweetUnlocker.exe
2) Selecione a opção: 1
3) Aguarde a extração terminar.
4) O mod será extraído para a pasta: SweetRlk
